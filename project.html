Eye Blinking Morse Code

Morse Code

A hands-free communication system that converts eye blinks into Morse code using computer vision techniques.

Technologies Used: Python, OpenCV, Dlib, Eye Aspect Ratio (EAR)

Functionality: Detects blink duration from webcam feed to identify dots and dashes in Morse code

Purpose: Helps individuals with speech or mobility impairments communicate using only eye movements

Features: Real-time blink detection, Morse code translation, customizable blink thresholds

Accuracy: Achieved 92% blink detection and 88% Morse code translation accuracy
